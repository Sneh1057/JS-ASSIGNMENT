Procedure of connecting JS file with HTML file :

We can include JavaScript in our HTML in three ways :

  • Embedding Code
  • Inline Code
  • External File

Embedding Code

To add the JavaScript code into the HTML pages, we can use the script tag of the HTML that wrap around JavaScript code inside the HTML program.

We can also define JS code in body tag or head tag.

Inline Code

Generally, this method is used when we have to call a function in the HTML event attributes.

There are many cases (or events) in which we have to add JavaScript code directly eg., OnMover event, OnClick, etc.

External File

We can create external JavaScript file and embed it in many html page.

It provides code re usability because single JavaScript file can be used in several html pages.

An external JavaScript file must be saved by .js extension. It is recommended to embed all JavaScript files into a single file. It increases the speed of the webpage.
